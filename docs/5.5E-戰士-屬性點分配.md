# 5.5E戰士分配屬性點

## 生成你的屬性值

你使用以下三種方法之一決定屬性值。 你的DM可能會需要你使用其中某一特定方法。

- 標準陣列。 用以下六個屬性值來表示你的屬性：15、14、13、12、10、8。
- 隨機生成。 擲四枚 d6 並記下最高三個骰值之和。 之後再重複五次，得到共計六個數位。
- 購點。 你有27個點數用來購買屬性值。 每項屬性值的花費點數都顯示在屬性值花費點數表中。 例如，得 到一個14點的屬性值會消耗27點數中的7點。
